# 采样进行HSV空间的分割

## 通过图片进行采样

   01 运行hsv_pic_sample.py

   02 在originimg窗口中找到想要提取的区域，按‘w’

   03 然后通过鼠标进行慢滑区域

   04 再按一下‘w’并且移开鼠标按‘q’

   05 得到结果

   ### note：红色的颜色有两个区域所以得叠加,观察法好吧

   红色：更改字典test0和test1

   其他颜色：更改字典test2

   其他：可以在riginimg窗口中按‘s’保存图片
